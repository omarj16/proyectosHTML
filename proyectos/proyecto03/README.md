# Ejercicio de un Carrusel de Imagenes sin Javascript

Este ejemplo muestra una estructura HTML básica para un carrusel de imágenes de tres elementos y los estilos CSS correspondientes para crear un carrusel visualmente atractivo y funcional sin utilizar JavaScript.

## Estructura del Proyecto

La estructura del proyecto es simple y consta de archivos HTML y CSS:

Proyecto/
|-- index.html
|-- styles.css
|-- img/
|   |-- ashish-kumar-senapati-FTUSP0ZH49I-unsplash.jpg
|   |-- matt-barrett-uf-HwVJBDlQ-unsplash.jpg
|   |-- simon-wilkes-uH5OXBhnSNg-unsplash.jpg
|-- README.md


## Uso

Abre el archivo `index.html` en tu navegador para ver el carrusel de imagenes sin javascript. Puedes personalizar el contenido y los estilos según tus necesidades.

## Estilos CSS

El archivo `styles.css` contiene los estilos para el carrusel y la página en general. Siéntete libre de ajustar estos estilos para adaptarlos a tus preferencias.
